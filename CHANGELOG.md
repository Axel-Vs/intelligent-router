### Added

- **Saved Runs Tab**: Complete optimization history with management tools
  - Browse all past optimization runs in comparison table
  - Three action buttons per run: View Map, Input Data CSV, Route Solution CSV
  - Re-optimization capability: load past runs back to Optimizer
  - Delete functionality to remove unwanted runs
  - Selection controls for future batch operations
- **Route Solution CSV Export**: Comprehensive 14-column detailed route solution
  - Route Number, Stop Sequence, Stop Type (pickup/delivery)
  - Vendor details: ID, Name, City, Address
  - Recipient details: Name, City, Address
  - Cargo: Weight (kg), Volume (m³)
  - Scheduling: Requested Delivery Date, Requested Loading Date
- **Clickable Branding**: Click "Intelligent Router" title to navigate to Optimizer tab
- **Black Tab Navigation**: Modern dark tabs with white active highlights for better contrast

### Changed

- **Button Naming**: Renamed buttons for clarity
  - "Download" → "Input Data CSV"
  - "Solution" → "Route Solution CSV"
  - Added "View Map" button for direct map access
- **Tab Styling**: Changed tab navigation to black background (#000000) with white active tab highlights
- **Button Layout**: Increased action button width to 320px with 9px font for longer labels
- **Title Interaction**: Added hover effect and pointer cursor to platform title

### Fixed

- **Period Variable Error**: Fixed UnboundLocalError when re-running past optimizations
  - Added period = None initialization at function start
  - Works for both CSV and JSON optimization requests
- **Column Mapping**: Fixed KeyError for latitude, longitude, name when reloading past runs
  - Added conditional checks before all field mappings
  - Handles both fresh uploads and CSV reloads seamlessly
- **CSV Path Issue**: Fixed past run loading to use full path instead of filename
  - Changed from filename-only to full results/runs/{runId}/input.csv path
  - Optimizer can now correctly read CSV files from past runs
